API Keys

Your own keys power your generations. Stored on the server; for security we never show them again after saving.
How to get a free YouTube Data API key
  1. Open console.cloud.google.com and create or pick a project.
  2. APIs & Services → Library → search YouTube Data API v3 → Enable.
  3. APIs & Services → Credentials → Create credentials → API key.
  4. Paste the key (AIza…) above. Free tier = 10,000 units/day.
Validates new creators and gives accurate view counts on your scrapes — on your own quota, so it never collides with other members.
How to get a Gemini API key
  1. Open aistudio.google.com/apikey and sign in.
  2. Click Create API key (pick a project or let it make one).
  3. Paste the key (AIza…) above.
Powers thumbnail & image generation on your own key, so heavy use never rate-limits other members.
AWS S3 — video hosting
Finished videos you upload (on the Publish page or a card’s Video box) go straight to your own S3 bucket (big files never touch this server). Access key & secret are stored write-only; bucket & region you can see here.
How to set up an S3 bucket + key
  1. In the S3 console create a bucket (note its name + region).
  2. Add a CORS rule allowing PUT/GET from this site so the browser can upload direct.
  3. In IAM create a user with s3:PutObject/s3:GetObject on that bucket, then create an access key.
  4. Paste the access key ID + secret + bucket + region above.
Leave blank to use the channel default while testing.
